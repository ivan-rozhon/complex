# [Complex]

Experimental web dev pack, that contains:
- NoSQL PHP CMS
- JS Admin App (Angular)
- REST API
- Local development environment

## Install
```
npm install
bower install
```

## Build
Basic build & watch:
```
gulp
```
or separately:
```
gulp build
gulp watch
```
production:
```
gulp build -prod
```

## Local dev server
(PHP ~7.1.0 required)
```
gulp connect
```
