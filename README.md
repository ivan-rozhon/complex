# [Complex]

Experimental web dev pack, that contains:
- PHP and Javascript (Angular) CMS
- REST API
- Local development environment (Webpack, PHP Webserver)

## Install
```
npm install
```

## Build & watch
Basic build & watch:
```
npm run build
```

## Local dev server
(PHP ~7.1.0 required)

After successful build (in separate terminal):
```
npm run serve
```
